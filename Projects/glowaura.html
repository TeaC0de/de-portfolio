GlowAura – Shopify Beauty Store

A Shopify-based beauty store built with my own concept and branding during a guided course project.

Status

🛠️ Currently in progress — I plan to reactivate the Shopify store to finalize the design, take screenshots and videos, and complete the visual walkthrough.

Project Overview

GlowAura is a concept store built with the Shopify platform. While the course example focused on a t-shirt shop, I created my own store using different branding, colors, and products. I explored the admin tools, store setup, discount settings, and learned how to build an e-commerce flow from scratch.

What I Worked On

  • Set up the storefront layout and navigation
  • Added products, collections, categories, and images
  • Created a 15% “Exclusive Partner” discount
  • Styled the theme with Shopify’s built-in tools
  • Configured basic settings like shipping, policies, and payment setup

What I Plan to Do Next

  • Reactivate the store temporarily to gather visuals
  • Create a walkthrough with screenshots and highlights
  • Explore custom styling with CSS and JavaScript
  • Add subtle visual improvements that reflect the soft, feminine branding of the shop
